Instalar Firebase CLI:\n```bash\nnpm install -g firebase-tools\nfirebase login\n```\n\n### 2. Inicializar Functions en tu proyecto:\n```bash\ncd D:\\Proyectos\\CantinaDelCharro\nfirebase init functions\n```\n\n### 3. Copiar el código de `FIREBASE_FUNCTIONS_SMS.js` a `functions/index.js`\n\n### 4. Instalar dependencias:\n```bash\ncd functions\nnpm install firebase-functions firebase-admin\n```\n\n### 5. Desplegar Functions:\n```bash\nfirebase deploy --only functions\n```\n\n---\n\n## 🔧 PASO 2: Configurar Proveedor SMS Real\n\n### Opción A: Twilio (Recomendado)\n```bash\ncd functions\nnpm install twilio\n```\n\n**Agregar a functions/index.js:**\n```javascript\nconst twilio = require('twilio');\nconst client = twilio('ACCOUNT_SID', 'AUTH_TOKEN');\n\n// Reemplazar la simulación con:\nawait client.messages.create({\n  body: `🍺 La Cantina del Charro\\n\\nTu código: ${otpCode}\\n\\nVálido por 5 min.`,\n  from: '+1234567890', // Tu número Twilio\n  to: phoneNumber\n});\n```\n\n### Opción B: AWS SNS\n```bash\nnpm install aws-sdk\n```\n\n### Opción C: SendGrid\n```bash\nnpm install @sendgrid/mail\n```\n\n---\n\n## 📱 PASO 3: Actualizar URL en la App\n\n**En `src/services/authService.js`:**\n```javascript\nconst FIREBASE_FUNCTIONS_URL = 'https://us-central1-TU-PROYECTO-ID.cloudfunctions.net';\n```\n\n**Reemplaza `TU-PROYECTO-ID` con tu ID real de Firebase.**\n\n---\n\n## 🧪 PASO 4: Probar el Sistema\n\n### Modo Desarrollo (sin SMS real):\n1. Las Functions devuelven el código en la respuesta\n2. Puedes ver el código en los logs de Firebase\n3. Perfecto para testing\n\n### Modo Producción (SMS real):\n1.
